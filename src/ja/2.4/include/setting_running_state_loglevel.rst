#. | Exastro システムにシステム管理者でログインします。

#. | メニューより :menuselection:`システム状態設定` をクリックします。

   .. image:: /images/ja/manuals/platform/settings_running_state/settings_running_state_menu.png
      :width: 200px
      :align: left
